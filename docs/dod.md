# Definition of Done (DoD)

## MVP‑фичи
- Реализованы пользовательские сценарии без критичных дефектов.
- Покрыты базовыми тестами ключевые модули (чтение, импорт, синхронизация, AI‑summary).
- Документация обновлена (минимум: README и docs/spec_mvp.md).
- Сборка проходит на CI, нет блокирующих ошибок; команды CI воспроизводимы локально.
- Наблюдаемость на минимальном уровне: логирование ошибок и метрики доступности.
- PR прошел ревью согласно docs/workflow.md (минимум 1 аппрув, без self‑merge).
- Артефакты проектной документации сохранены в репозитории (docs/*.md и актуальные диаграммы).

## Backend и данные
- Миграции БД описаны и воспроизводимы.
- Схемы API согласованы и задокументированы (OpenAPI или эквивалент).
- Хранилище файлов и политика доступа проверены.

## Клиент
- Производительность чтения: перелистывание/скролл без фризов, 95‑й перцентиль задержки ≤ 300 мс на типовом устройстве.
- Работа без сети не приводит к потере данных; синхронизация догоняет.

## Релиз
- Версионирование и changelog обновлены.
- Проведена smoke‑проверка на целевых платформах.
